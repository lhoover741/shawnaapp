# Ravishing Beauté Replit Blueprint

This is a complete React/Vite app blueprint for the **Ravishing Beauté** website/app. It is built for Replit first, then can be pushed to GitHub and deployed on Cloudflare Pages.

## What is included

- Premium mobile-first salon website/app
- Pages: Home, Gallery, Services / Pricing, About Your Stylist, Booking, Reviews, Contact, Policies
- Real local image assets included in `src/assets/images`
- Service price logic in `src/data/services.js`
- Booking validation and booking summary logic in `src/utils/bookingLogic.js`
- Native SMS/contact handoff using `sms:` links
- No Instagram embeds
- Clean review presentation with approved/curated review data

## Business rules included

- $25 deposit required
- Hours: 8:30 AM to 6:00 PM by appointment
- Closed Sunday and Monday
- Same-day bookings only if approved
- Braiding hair included only in natural colors: 1, 1B, 2, 4
- Policies include: “Please come detangled to stay on schedule.”
- Policies do not say detangling is free

## Replit setup

1. Upload this zip into Replit or import it as a project.
2. Run `npm install`.
3. Run `npm run dev`.
4. Test the booking page, service selector, add-ons, hair color notes, and SMS contact button.
