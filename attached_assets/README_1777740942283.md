# Ravishing Beauté / shawna-app Replit Build

This is a Replit-ready React + Vite app package for the Ravishing Beauté website/app.

## What this version includes

- Premium mobile-first Ravishing Beauté layout
- Home, Gallery, Services / Pricing, About Your Stylist, Booking, Reviews, Contact, and Policies
- Updated gallery/reviews direction from the newest project source
- Native polished layouts instead of Instagram embeds
- Booking request flow that opens the phone/text app with a prefilled appointment message
- Contact question form that opens the phone/text app with a prefilled message
- Service pricing logic in `src/data/services.js`
- Deposit, estimated total, add-ons, and remaining balance calculation
- Business rules:
  - $25 deposit required
  - Hours: 8:30 AM to 6:00 PM by appointment
  - Closed Sunday and Monday
  - Same-day bookings only if approved
  - Braiding hair included only in natural colors 1, 1B, 2, and 4 unless otherwise specified
  - “Please come detangled to stay on schedule.”

## Replit instructions

1. Upload this zip to Replit or import into a new Replit project.
2. Run `npm install`.
3. Run `npm run dev`.
4. Test the Booking and Contact buttons from a phone/mobile preview for SMS behavior.

## GitHub / Cloudflare notes

This build is ready for GitHub and Cloudflare Pages style deployment. For Cloudflare Pages, use:

- Build command: `npm run build`
- Output directory: `dist`
